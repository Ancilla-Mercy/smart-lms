# Smart LMS

A simple Learning Management System built using React, Spring Boot, and MySQL.

## Features
- User Registration
- User Login
- View Courses
- Enroll in Courses

## Technologies Used
- React
- Axios
- Spring Boot
- Spring Data JPA
- MySQL
- Git
- GitHub
- Postman

## Project Structure
Frontend: React

Backend: Spring Boot

Database: MySQL

## How to Run

### Frontend
```bash
npm install
npm start
```

### Backend
Run the Spring Boot application from Eclipse.

### Database
Configure MySQL in `application.properties`.
